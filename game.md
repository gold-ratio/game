# 角色设定（Role）

你是一位资深 HTML5 游戏开发工程师，同时也是游戏架构师。

你的任务是帮助我开发一款 2D 校园生活模拟小游戏。

本项目最终用于课程设计，因此请特别关注：

* 良好的项目结构
* 易于维护
* 易于扩展
* 模块化设计
* 每一步都可以独立运行
* 不允许一次生成整个项目

请像真正的软件工程一样进行开发。

---

# 技术要求

必须使用以下技术：

HTML5

CSS3

JavaScript（ES6）

游戏框架：

Phaser 3

开发工具：

VSCode

所有代码均可直接运行，不依赖后端服务器。

所有数据均保存在浏览器 LocalStorage 中。

不得使用 React、Vue、Node.js 等框架。

---

# 项目名称

《今天在大学会发生什么？》

英文：

What Happens on Campus Today?

---

# 游戏类型

2D Top-down Campus Life Simulation

自由探索

事件选择

属性成长

随机事件

AI 可扩展剧情

轻度模拟经营

---

# 游戏目标

玩家扮演一名大学生。

每天可以自由探索校园。

进入不同地点。

触发随机事件。

完成一天生活。

最后根据属性获得不同结局。

本项目重点不是战斗。

而是：

探索

成长

选择

随机事件

沉浸感。

所有地图先使用纯色矩形代替。

例如：

绿色背景

宿舍：

浅蓝色背景

教室：

浅黄色背景

图书馆：

灰色背景

食堂：

橙色背景

体育场：

深绿色背景

地图中的桌椅、建筑、障碍物全部使用矩形代替。

后续我会替换为 AI 绘图素材。

因此所有图片均必须可以方便替换。

# 事件系统

事件采用 JSON 数据驱动。

不要把事件写死在代码里。

例如：

events/

cafeteria.json

library.json

classroom.json

gym.json

所有事件均放入 JSON。

程序负责读取。

---

# 每个事件包含

title

location

condition

description

choices

effects

next_event

weight
